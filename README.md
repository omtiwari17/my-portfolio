## 📌 About

Personal portfolio website built from scratch with **HTML5, CSS3, and vanilla JavaScript** — no frameworks, no templates. Designed to be fast, fully responsive, and visually clean across all screen sizes and devices.

---

## ✨ Features

| Section | Description |
|---|---|
| 🏠 **Home** | Hero intro with a quick personal overview |
| 🎓 **Education & Skills** | Academic background and technical skill showcase |
| 💼 **Experience** | Internship and work highlights |
| 🛠️ **Projects** | Selected projects with images, tech stacks, and descriptions |
| 📄 **Resume** | One-click downloadable PDF |
| 📬 **Contact** | Simple contact form to reach out directly |
| 🌗 **Dark / Light Mode** | Toggle for comfortable viewing in any environment |

---

## 🛠️ Tech Stack

- **Markup & Logic:** HTML5, CSS3, JavaScript (ES6+)
- **Styling:** Custom CSS with animations, transitions, and responsive layouts
- **Icons:** Font Awesome
- **Fonts:** Google Fonts
- **Hosting:** GitHub Pages

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/omtiwari17/my-portfolio.git

# Open locally — no build step needed
cd my-portfolio
open index.html   # or just open in your browser
```
